# /trend-report — 트렌드 키워드 리포트

## 개요

네이버, 구글 트렌드, DC인사이드에서 트렌드 키워드를 수집하고 심소 브랜드에 적합한 키워드로 클러스터링합니다.

**관련 에이전트**: `trendy-keyword-reporter` 에이전트가 이 스킬을 기본 도구로 활용하여 더 심화된 분석을 수행할 수 있습니다.

## 실행 절차

### Step 1: 의존성 확인 & 설치
```bash
pip3 install -r .claude/skills/trend-report/scripts/requirements.txt --quiet
```

### Step 2: 키워드 수집
```bash
python3 .claude/skills/trend-report/scripts/collect_all.py
```
이 스크립트는 네이버, 구글 트렌드, DC인사이드에서 트렌드 키워드를 수집하여 JSON으로 출력한다.

### Step 3: 수집된 키워드 분석 & 클러스터링

수집된 JSON 결과를 아래 기준으로 분석한다:

#### 심소 적합성 판단 기준
심소에 어울리는 키워드:
- 일상, 소확행, 슬로우라이프, 미니멀리즘 관련
- 계절감, 날씨, 자연, 산책 관련
- 요리, 독서, 차, 취미 관련
- 비움, 정리, 디톡스 관련
- 마음 건강, 여유, 힐링 관련, 게임
- 주제에 상관없는 취미생활

심소에 어울리지 않는 키워드 (제외):
- 정치, 선거, 정당
- 연예인 스캔들, 자극적 뉴스
- 주식, 코인, 투자
- 경쟁, 승부
- 혐오, 논쟁, 갈등

#### 클러스터 카테고리
수집된 키워드를 다음 카테고리로 분류:
1. **계절·날씨** — 현재 계절 관련 키워드
2. **음식·요리** — 레시피, 카페, 차, 간식
3. **자연·산책** — 꽃, 나무, 공원, 날씨
4. **마음·비움** — 힐링, 정리, 미니멀, 디톡스
5. **취미·기록** — 독서, 필사, 그림, 사진
6. **트렌드 활용** — 심소 톤으로 재해석 가능한 일반 트렌드

### Step 4: 보고서 작성

분석 결과를 `docs/trend-report.md`에 작성한다. 형식:

```markdown
# 트렌드 키워드 리포트

> 수집일: {날짜}
> 소스: 네이버, 구글 트렌드, DC인사이드

## 수집 요약
- 총 수집 키워드: N개
- 심소 적합 키워드: N개

## 심소 적합 클러스터

### 계절·날씨
| 키워드 | 소스 | 활용 아이디어 |
|--------|------|--------------|
| ... | ... | ... |

### 음식·요리
...

## 콘텐츠 제안
수집된 트렌드 기반으로 심소에서 활용할 수 있는 콘텐츠 아이디어 3~5개 제안.
(챌린지 주제, 오늘의 한 마디, 계절 게시판 주제 등)

## 제외된 키워드
심소 정책상 부적합한 키워드 목록 (정치, 자극적 뉴스 등)
```
