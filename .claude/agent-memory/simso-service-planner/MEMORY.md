# 심소(SIMSO) 서비스 기획 메모리

## 서비스 정체성
- "심플한 소소함" = 느리게 읽고, 작게 기록하고, 조용히 공감하는 커뮤니티
- 핵심 키워드: 소확행, 디지털 미니멀리즘, 슬로우 라이프, 비움과 정리
- 슬로건 후보: "오늘도 소소하게.", "작은 하루가 모여 좋은 삶이 됩니다."

## 톤앤매너
- "~하면 좋겠어요" 형태의 제안형 어미
- 공지도 편지처럼 작성
- 명령/강요 없이 초대하는 느낌, 이모지 자제

## 핵심 금지 사항
- AI 생성 콘텐츠 게시 금지 (운영 뒷단에서만 AI 활용)
- 정치/종교/혐오/광고 금지
- 숫자 비노출 (팔로워 수, 게시물 수, 좋아요 수 등)
- 과시/비교 문화 지양

## 주요 기능 (concept.md 기준)
- 오늘의 소소함: 1일 1글 제한, 300자, 사진 1장, 공감해요 리액션
- 계절 게시판: 비움 일지, 산책 기록, 소소 레시피, 고요한 취미
- 편지함: 랜덤 매칭, 주 1통, 비동기
- 조용한 챌린지: 자기 기록 중심, 비경쟁
- 피드: 관심 키워드 60% + 다양한 관심사 40%

## 콘텐츠 기획 (content-plan.md)
- 기존 기능 위 콘텐츠: 오늘의 한 마디(글감), 계절 편지, 편지 주제, 챌린지 라인업
- 신규 콘텐츠: 계절 서랍, 소소한 발견 카드, 고요한 추천, 빈 페이지, 계절 마무리 질문
- 운영 사이클: 입춘/입하/입추/입동 기준 계절 전환
- 주간 루틴은 의도적으로 단순하게 유지 (조용한 앱 원칙)

## 기술 스택 요약
- Next.js 15 (App Router) + TypeScript + Tailwind
- 모바일: React Native (Expo) WebView 래핑
- DB: PostgreSQL (Neon), Redis (Upstash), Cloudflare R2
- AI 파이프라인: Claude API (금지 소재/논조), AI 생성물 탐지, 이미지 안전성
- 배포: Vercel, Turborepo 모노레포
