- 패션위크 같은 일반 트렌드를 심소 톤으로 재해석하는 전략이 더 효과적
- 현재 소스는 전면 교체 필요 (라이프스타일 중심 소스로)

### 소스별 개선 방향
- **현재 소스 유지 불필요**: 네이버 뉴스, DC인사이드는 심소 적합도 0%로 제외 고려
- **우선 추가 소스**: 인스타그램 해시태그 (#일상, #힐링, #소확행, #슬로우라이프)
- **차순위 추가**: 네이버 블로그 라이프스타일 카테고리, 유튜브 브이로그 채널
- **시간대 최적화**: 주말 오전(10-12시), 평일 저녁(19-22시) 라이프스타일 검색 증가
- **계절 패턴**: 2월 말~3월 초 봄 관련 키워드 증가 예상 (환절기, 봄맞이, 옷장정리)

## 클러스터링 패턴

### 높은 심소 적합도 키워드 특징
- 계절감 있는 키워드 (봄, 환절기, 벚꽃, 단풍)
- 일상 활동 키워드 (산책, 정리, 기록, 요리)
- 감성 표현 키워드 (힐링, 여유, 소소한, 따뜻한)
- 느린 속도, 소소한 행복 관련 (차 마시기, 독서)
- 비움·정리 관련 (미니멀, 디톡스, 옷장정리)

### 낮은 심소 적합도 키워드 특징
- 실시간 뉴스성 키워드 (정치, 사회 이슈)
- 자극적/논쟁적 키워드 (스캔들, 사건사고)
- 경쟁/성과 중심 키워드 (투자, 주식, 스포츠)
- 커뮤니티 전용 은어 (갤러리명, 약칭)

### 트렌드 재해석 전략
- 일반 트렌드를 심소 톤으로 재해석하는 것이 효과적
- 예: 패션위크 → 옷장 정리, 미니멀 스타일, 나다움 찾기
- 예: 올림픽 → 응원하는 마음, 함께 보는 즐거움 (단, 경쟁 강조는 지양)

## 다음 작업 개선 사항

2. 정치/경제 키워드 블랙리스트 필터링 적용
3. 계절 키워드 가중치 시스템 구축

### 중기 개선 (2-4주)
1. 네이버 블로그 API 활용 스크립트 작성
2. 유튜브 라이프스타일 채널 키워드 추출
3. 시간대별 수집 스케줄링 (주말 오전/평일 저녁)
4. 심소 적합도 자동 점수화 알고리즘 (1-10점)

### 장기 전략 (1개월 이상)
1. 계절별 키워드 사전 구축 및 자동 매핑

3. 롱테일 키워드 모니터링 (지속 검색어 추적)
