# 심소(SIMSO) -- 콘텐츠 기획서

> "작은 하루가 모여 좋은 삶이 됩니다."

---

## 1. 콘텐츠 기획 원칙

심소의 모든 콘텐츠는 다음 다섯 가지 원칙 위에 설계합니다.

| 원칙 | 설명 | 반대로 말하면 |
|------|------|--------------|
| **자기 기록 중심** | 남에게 보여주기 위한 게 아니라, 나를 위해 남기는 기록 | 타인의 시선을 의식한 퍼포먼스가 아님 |
| **느린 참여** | 매일 하나, 한 주에 하나 -- 천천히 쌓이는 경험 | 실시간 반응이나 빠른 소비를 유도하지 않음 |
| **비경쟁** | 순위, 랭킹, 비교 요소를 철저히 배제 | 좋아요 수, 참여율 수치를 노출하지 않음 |
| **계절 감각** | 자연의 리듬에 맞춰 콘텐츠가 흐름 | 트렌드나 이슈에 반응하는 콘텐츠가 아님 |
| **여백의 가치** | 콘텐츠가 없는 시간도 의미 있게 설계 | 빈 공간을 억지로 채우지 않음 |

### 콘텐츠 톤앤매너

- "~하면 좋겠어요", "~해보는 건 어떨까요" 형태의 제안형 어미
- 명령이나 강요 없이, 초대하는 느낌
- 문장은 짧고 담백하게
- 감탄사, 이모지, 과한 수식어 자제
- 공지도 편지처럼

**톤 예시:**

| 상황 | 심소 톤 | 지양하는 톤 |
|------|---------|-----------|
| 콘텐츠 안내 | "이번 주는 하늘을 올려다보면 좋겠어요." | "이번 주 미션! 하늘 사진 GO!" |
| 참여 유도 | "생각나면 적어두면 좋겠어요." | "지금 바로 참여하세요!" |
| 챌린지 완료 | "꾸준히 기록하셨네요. 고마워요." | "축하합니다! 챌린지 완료!" |
| 비활동 사용자 | "심소는 여기 있어요. 언제든 괜찮아요." | "오랜만이에요! 놓친 콘텐츠를 확인하세요!" |

## 2. 기존 기능 위에 얹는 콘텐츠

### 2.1 오늘의 소소함 -- 글감 제안 "오늘의 한 마디"

**개요:** 매일 아침, 메인 피드 상단에 짧은 글감 한 줄이 표시됩니다. 글을 쓰고 싶은데 무슨 이야기를 할지 모르겠을 때, 가볍게 꺼내볼 수 있는 실마리입니다.

**설계:**

- 하루에 하나, 자정 기준으로 교체
- 사용자가 쓴 글과 직접 연결되지 않음 (태그나 분류 없음)
- 반드시 따라야 하는 주제가 아니라, 스쳐가는 영감

**글감 예시:**

| 계절 | 글감 |
|------|------|
| 봄 | "오늘 처음 본 꽃이 있었나요?" |
| 여름 | "요즘 자주 듣는 소리가 있다면요." |
| 가을 | "올해 안에 꼭 하고 싶은 작은 일이 있나요?" |
| 겨울 | "따뜻했던 순간 하나를 떠올려보면 좋겠어요." |
| 공통 | "오늘 먹은 것 중 제일 맛있었던 건 뭐였나요?" |
| 공통 | "지금 창 밖에 뭐가 보이나요?" |
| 공통 | "아무 이유 없이 기분이 좋았던 순간이 있었나요?" |

**운영 방식:**

- 초기: 기획팀이 계절별 글감 풀(pool)을 미리 작성 (계절당 90개, 총 360개)
- 순환 배치하되, 같은 글감이 가까운 시기에 반복되지 않도록 간격 관리
- 사용자가 글감을 제안할 수 있는 "글감 우체통" 기능을 2차에 검토
- 글감에 반응한 사용자 수 등 수치는 어디에도 노출하지 않음

---

### 2.2 계절 게시판 -- 계절 전환 콘텐츠 "계절 편지"

**개요:** 계절이 바뀔 때마다 심소가 사용자에게 보내는 짧은 편지입니다. 지나간 계절을 돌아보고, 다가오는 계절을 맞이하는 부드러운 전환점 역할을 합니다.

**설계:**

- 연 4회 발행 (입춘, 입하, 입추, 입동 기준)
- 게시판 상단에 고정, 7일간 노출 후 아카이브
- 이전 계절의 게시판 글들 중 인상적인 문장 몇 개를 (작성자 동의 하에) 익명으로 인용

**계절 편지 예시 (가을에서 겨울로):**

> 가을 동안 심소에는 많은 산책 기록이 쌓였어요.
> 누군가는 은행나무 아래를 걸었고, 누군가는 빗소리를 들으며 걸었어요.
>
> "낙엽 밟는 소리가 좋아서 일부러 낙엽 위로 걸었어요." -- 어느 사용자의 산책 기록에서
>
> 이제 겨울이에요.
> 조금 더 따뜻하게 입고, 조금 더 느리게 걸어도 괜찮아요.
> 이번 겨울에는 어떤 소소함을 남기게 될까요.

**운영 방식:**

- 기획팀이 직접 작성 (AI 생성 불가 원칙)
- 인용할 문장은 작성자에게 앱 내 알림으로 동의 요청
- 동의 받은 문장만 익명으로 인용, 원문 수정 없이 그대로 사용

---

### 2.3 편지함 -- 계절 주제 편지 "계절 편지 주제"

**개요:** 랜덤 매칭 편지를 쓸 때, 무슨 이야기를 꺼내야 할지 막막한 사용자를 위해 계절마다 가벼운 편지 주제를 제안합니다.

**설계:**

- 편지 작성 화면에 "이런 이야기는 어떨까요?" 영역으로 노출
- 주제를 선택해도 되고, 무시해도 됨 (강제 아님)
- 계절별로 3개씩, 총 12개 주제를 순환

**편지 주제 예시:**

| 계절 | 주제 |
|------|------|
| 봄 | "요즘 새로 시작한 게 있나요?" |
| 봄 | "좋아하는 봄 음식이 있으면 알려주세요." |
| 봄 | "올봄에 가보고 싶은 곳이 있나요?" |
| 여름 | "더울 때 자주 하는 일이 있나요?" |
| 여름 | "기억에 남는 여름이 있다면요." |
| 여름 | "요즘 마시는 음료가 있으면 알려주세요." |
| 가을 | "가을이 오면 떠오르는 것이 있나요?" |
| 가을 | "요즘 읽고 있는 책이 있다면요." |
| 가을 | "걷기 좋은 길을 알고 있으면 알려주세요." |
| 겨울 | "올해 고마웠던 순간이 있나요?" |
| 겨울 | "겨울에 자주 만들어 먹는 게 있다면요." |
| 겨울 | "새해에 하고 싶은 작은 일이 있나요?" |

---

### 2.4 조용한 챌린지 -- 정기 콘텐츠 라인업

**개요:** 조용한 챌린지에 올릴 콘텐츠 라인업입니다. 챌린지는 경쟁이 아니라 자기 기록이므로, 완주 여부를 타인에게 노출하지 않으며, 중간에 빠져도 아무 패널티가 없습니다.

**챌린지 유형 분류:**

| 유형 | 기간 | 성격 |
|------|------|------|
| **주간 챌린지** | 7일 | 가볍게 시작, 일상 관찰형 |
| **월간 챌린지** | 30일 | 습관 형성형, 꾸준한 기록 |
| **계절 챌린지** | 약 90일 | 긴 호흡의 변화, 비움/정리 중심 |

**주간 챌린지 예시 (각 계절별 2개씩):**

| 계절 | 챌린지 | 기록 방식 |
|------|--------|----------|
| 봄 | "일주일 동안 매일 하늘 사진 찍기" | 사진 1장 + 한 줄 |
| 봄 | "매일 처음 보는 것 하나 기록하기" | 텍스트 |
| 여름 | "일주일 동안 매일 물 한 잔 마시며 멈추기" | 텍스트 (그 순간 떠오른 생각) |
| 여름 | "저녁 하늘 색깔 기록하기" | 사진 1장 + 색깔 이름 |
| 가을 | "매일 고마운 것 하나 적기" | 텍스트 |
| 가을 | "일주일 동안 같은 길 걸으며 변화 기록하기" | 사진 1장 + 한 줄 |
| 겨울 | "매일 따뜻했던 순간 하나 기록하기" | 텍스트 |
| 겨울 | "잠들기 전 오늘 하루를 한 문장으로 정리하기" | 텍스트 |

**월간 챌린지 예시:**

| 챌린지 | 설명 |
|--------|------|
| "한 달간 하루 한 끼 직접 만들어 먹기" | 소소 레시피 게시판과 연계 가능 |
| "30일 동안 매일 10분 산책하기" | 산책 기록 게시판과 연계 가능 |
| "한 달간 하루에 한 문장 필사하기" | 고요한 취미 게시판과 연계 가능 |
| "매일 안 쓰는 물건 하나 비우기" | 비움 일지 게시판과 연계 가능 |

**계절 챌린지 예시:**

| 챌린지 | 설명 |
|--------|------|
| "이번 계절 동안 책 3권 천천히 읽기" | 완독이 아니라 읽는 과정을 기록 |
| "계절이 바뀔 때까지 옷장 한 칸 비우기" | 한 번에 다 하지 않고, 천천히 |
| "90일 동안 매주 한 가지 새로운 차 마셔보기" | 약 12-13가지 차를 경험 |

**챌린지 UX 원칙:**

- 참여 인원수 비노출
- 완주율, 달성률 등 수치 비노출
- "오늘은 못 했어요" 버튼 없음 -- 기록하지 않은 날은 그냥 비어 있을 뿐
- 챌린지 완료 시 배지나 보상 대신, 자신만 볼 수 있는 "기록 모아보기" 제공
- 다른 사용자의 챌린지 기록은 피드에서 자연스럽게 섞여 보임 (별도 챌린지 피드 없음)

## 3. 신규 콘텐츠 제안

### 3.1 "계절 서랍" -- 나만의 계절 아카이브

**개요:** 한 계절 동안 내가 남긴 모든 기록(오늘의 소소함, 게시판 글, 챌린지 기록)이 계절이 끝나면 하나의 "서랍"으로 정리되어 보관됩니다.

**사용자 가치:** 흩어진 일상 기록이 계절 단위로 묶이면, 3개월이라는 시간의 질감이 느껴집니다. "이번 가을에 나는 이런 사람이었구나"를 조용히 돌아볼 수 있습니다.

**설계:**

- 계절 종료 시 자동 생성 (입춘/입하/입추/입동 기준)
- 서랍 구성:
  - 내가 쓴 글 목록 (시간순)
  - 사진 모아보기 (격자형)
  - "이 계절의 한 마디" -- 내 글 중 가장 짧은 문장 하나를 자동 선택하여 표지에 표시
- 서랍은 나만 볼 수 있음 (비공개)
- 지난 계절 서랍은 언제든 다시 열어볼 수 있음

**심소 정체성 부합:**
- 기록의 축적 가치를 시각화
- 과시가 아닌 자기 회고 목적
- 숫자(글 수, 공감 수)가 아닌 내용 중심 구성

---

### 3.2 "소소한 발견" -- 관찰 기록 카드

**개요:** 산책 중, 출퇴근길에, 혹은 집 안에서 발견한 작은 것을 사진 한 장과 짧은 문장으로 남기는 가벼운 기록 형식입니다. "오늘의 소소함"보다 더 가볍고, 더 짧습니다.

**사용자 가치:** 하루를 글로 정리하기엔 부담스럽지만, 한 순간을 포착하고 싶을 때 사용합니다. "글을 쓴다"는 부담 없이 "발견을 남긴다"는 느낌으로 접근합니다.

**설계:**

- 사진 1장 + 50자 이내 텍스트 (오늘의 소소함의 300자보다 짧음)
- 하루 제한 없음 (발견은 여러 번 일어날 수 있으니까)
- 다만 피드에는 노출되지 않고, 내 프로필의 "발견 모아보기"에서만 확인 가능
- 원한다면 오늘의 소소함을 쓸 때 발견 카드를 첨부할 수 있음

**주의 사항:**
- 피드 노출이 없으므로 "좋아요 수집" 동기를 제거
- 순수하게 개인 기록 용도
- 오늘의 소소함(1일 1글)의 제한과 충돌하지 않는 별개 기능

---

### 3.3 "고요한 추천" -- 사용자 간 비공개 추천

**개요:** 좋은 글을 발견했을 때, 공개적인 공유 대신 조용히 한 사람에게만 전달할 수 있는 기능입니다.

**사용자 가치:** "이 글, 당신이 좋아할 것 같아요"라는 마음을 전하되, 바이럴이나 확산이 아닌 1:1의 조용한 공유입니다.

**설계:**

- 글 하단에 "조용히 전하기" 버튼
- 전달 대상: 편지를 주고받은 적 있는 사용자에게만 가능 (아무에게나 보내지 않음)
- 전달 시 짧은 메시지 추가 가능 (50자 이내): "이 글이 떠올랐어요."
- 전달 받은 사람에게는 "누군가 이 글을 전해주었어요"로 표시 (보낸 사람 닉네임은 포함)
- 하루 추천 횟수 제한: 1회 (남용 방지)
- 원글 작성자에게는 "누군가 당신의 글을 소중히 전했어요" 알림 (횟수 비노출)

**심소 정체성 부합:**
- 바이럴 공유가 아닌, 편지처럼 조용한 전달
- 숫자(공유 횟수) 비노출
- 편지함 기능과 자연스럽게 연결

---

### 3.4 "빈 페이지" -- 의도적 비활동 기능

**개요:** 오늘은 아무것도 쓰지 않겠다는 선택을 의미 있게 만드는 기능입니다. "오늘의 소소함"을 쓰지 않는 날에도, 빈 페이지 하나를 남길 수 있습니다.

**사용자 가치:** 아무것도 하지 않는 날도 괜찮다는 메시지를 서비스 차원에서 전합니다. 매일 무언가를 채워야 한다는 압박에서 벗어나게 합니다.

**설계:**

- 글 작성 화면에서 "오늘은 비워둘게요" 선택지
- 선택하면 내 계절 서랍에 빈 페이지로 기록됨 (날짜만 표시)
- 피드에 노출되지 않음 (나만 볼 수 있음)
- 빈 페이지에는 심소가 보내는 짧은 한 마디가 표시됨:
  - "쉬어가는 하루도 좋은 하루예요."
  - "비어 있어도 괜찮아요."
  - "가만히 있는 것도 하나의 기록이에요."
  - "오늘은 그냥, 오늘이에요."

**심소 정체성 부합:**
- "비움과 정리"라는 핵심 키워드와 직결
- 매일 기록해야 한다는 강박을 해소
- 빈 공간에도 가치를 부여하는 심소만의 방식

---

### 3.5 "계절 마무리 질문" -- 분기별 자기 회고

**개요:** 계절이 끝나갈 때, 심소가 사용자에게 3개의 질문을 보냅니다. 답하면 계절 서랍의 마지막 페이지에 저장됩니다.

**사용자 가치:** 한 계절을 의식적으로 마무리하는 경험을 제공합니다. 바쁘게 지나가는 시간 속에서 잠시 멈추고, "이번 계절은 나에게 어떤 의미였나"를 생각하게 합니다.

**질문 설계 (계절 공통 3문항):**

1. "이번 [계절]에 가장 기억에 남는 순간이 있나요?"
2. "이번 [계절]에 나에게 생긴 작은 변화가 있다면요."
3. "다음 [계절]의 나에게 한 마디 남긴다면요."

**운영 방식:**

- 계절 종료 7일 전에 알림 1회 발송: "이번 [계절]이 곧 끝나요. 마무리 질문이 도착했어요."
- 답변은 선택 사항 (3개 중 원하는 것만 답해도 됨)
- 답변은 비공개 (계절 서랍에만 저장)
- 답변 기한: 다음 계절 시작 후 7일까지 (총 14일)

## 4. 콘텐츠 운영 캘린더

연간 콘텐츠 운영의 흐름을 정리합니다.

### 연간 사이클

```
[입춘 2/4경]                    [입하 5/6경]
    |                              |
    v                              v
  봄 시작                         여름 시작

  - 봄 챌린지 오픈                 - 여름 챌린지 오픈
  - 겨울 마무리 질문 마감           - 봄 마무리 질문 마감
    |                              |
    | (봄 운영)                     | (여름 운영)
    | - 주간 챌린지 매주 교체         | - 주간 챌린지 매주 교체
    | - 글감 매일 교체               | - 글감 매일 교체
    | - 편지 주제 계절 반영           | - 편지 주제 계절 반영
    |                              |
    | [봄 종료 7일 전]               | [여름 종료 7일 전]
    | 마무리 질문 발송               | 마무리 질문 발송
    |                              |
    v                              v
[입추 8/7경]                    [입동 11/7경]
    |                              |
    v                              v
  가을 시작                        겨울 시작
  - 계절 편지 발행                 - 계절 편지 발행
  - 여름 서랍 생성                 - 가을 서랍 생성
  - 가을 챌린지 오픈               - 겨울 챌린지 오픈
  - 여름 마무리 질문 마감           - 가을 마무리 질문 마감
```

### 주간 루틴

| 요일 | 콘텐츠 | 비고 |
|------|--------|------|
| 매일 | 오늘의 한 마디 (글감) 교체 | 자정 기준 자동 |
| 월요일 | 주간 챌린지 시작 (새 챌린지 오픈) | 이전 챌린지와 겹쳐도 무방 |
| -- | 기록은 사용자 자율 | 요일별 강제 콘텐츠 없음 |

주간 루틴은 의도적으로 단순하게 유지합니다. 매일 새로운 이벤트를 만드는 것은 심소의 "조용한 앱" 원칙에 어긋납니다.

## 6. 콘텐츠 우선순위

MVP 및 이후 단계에 맞춰 콘텐츠 도입 순서를 정합니다.

### Phase 2 (핵심 기능과 함께)

| 콘텐츠 | 이유 | 구현 난이도 |
|--------|------|-----------|
| 오늘의 한 마디 (글감) | 메인 피드와 함께 제공, 초기 사용자의 첫 글쓰기 유도 | 낮음 (텍스트 풀 관리) |
| 빈 페이지 | 1일 1글 제한과 함께 작동, 쓰지 않는 선택도 존중 | 낮음 |

### Phase 3 (확장)

| 콘텐츠 | 이유 | 구현 난이도 |
|--------|------|-----------|
| 계절 편지 | 계절 게시판 오픈과 함께 | 낮음 (에디토리얼 콘텐츠) |
| 주간/월간 챌린지 라인업 | 조용한 챌린지 기능 구현 시 | 중간 (챌린지 엔진 필요) |
| 소소한 발견 카드 | 기록의 가짓수를 늘려 활성화 | 중간 |

### Phase 4 (커뮤니티)

| 콘텐츠 | 이유 | 구현 난이도 |
|--------|------|-----------|
| 계절 편지 주제 | 편지함 오픈과 함께 | 낮음 |
| 고요한 추천 | 편지함 사용자 간 연결 확장 | 중간 |
| 계절 서랍 | 1계절분 데이터 축적 후 의미 있음 | 중간 |
| 계절 마무리 질문 | 계절 서랍과 함께 | 낮음 |

## 7. 콘텐츠 금지 사항 재확인

모든 콘텐츠에 공통 적용되는 금지 원칙입니다. (concept.md 기준)

| 금지 항목 | 콘텐츠 기획 시 주의사항 |
|-----------|----------------------|
| AI 생성 콘텐츠 | 글감, 계절 편지, 챌린지 설명 등 모든 사용자 노출 텍스트는 사람이 직접 작성 |
| 정치/종교/혐오 | 글감 주제에 사회적 논쟁 소재를 포함하지 않음 |
| 경쟁/비교 유도 | 챌린지 참여 인원, 완주율, 랭킹 등 일체 비노출 |
| 과시 유도 | "자랑해보세요", "보여주세요" 류의 표현 사용 금지 |
| 소비 자극 | 챌린지 보상으로 할인 쿠폰, 포인트 등 상업적 보상 제공 금지 |
| 알림 남용 | 콘텐츠 관련 알림은 계절 전환 시 1회, 마무리 질문 시 1회로 제한 |

## 8. 성과 측정 (내부 지표, 사용자 비노출)

콘텐츠의 건강함을 판단하기 위해 내부적으로만 추적하는 지표입니다. 사용자에게는 어떤 수치도 노출하지 않습니다.

| 지표 | 측정 대상 | 목적 |
|------|----------|------|
| 글감 활용률 | 글감이 교체된 날 작성된 글 중 글감과 관련된 글의 비율 (AI 분석) | 글감의 영감 효과 확인 |
| 챌린지 지속률 | 챌린지 시작 후 3일차, 7일차에 기록을 남긴 비율 | 챌린지 난이도/매력도 판단 |
| 빈 페이지 사용률 | 빈 페이지를 선택한 사용자 비율 | 기능 인지도 및 필요성 확인 |
| 계절 서랍 열람률 | 서랍 생성 후 30일 내 열어본 비율 | 아카이브 가치 검증 |
| 편지 주제 선택률 | 편지 작성 시 제안 주제를 활용한 비율 | 주제의 유용성 판단 |
| 계절 마무리 답변률 | 마무리 질문에 1개 이상 답변한 비율 | 회고 콘텐츠의 참여도 확인 |
