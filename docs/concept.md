# 심소(SIMSO) — 커뮤니티 기획 초안

> "소소한 일상 & 단순한 삶으로의 회귀"

---

## 1. 컨셉

**심소** = 심플한 소소함

빠르고 자극적인 콘텐츠가 넘치는 시대에,
느리게 읽고 · 작게 기록하고 · 조용히 공감하는 커뮤니티.

### 핵심 키워드
- 소확행(소소하지만 확실한 행복)
- 디지털 미니멀리즘
- 슬로우 라이프
- 비움과 정리

### 우리가 아닌 것
- SNS식 좋아요 경쟁
- 과시·비교 문화
- 정보 과잉 피드
- AI가 만든 글·그림이 사람 행세하는 공간
- 정치·종교·혐오 등 갈등을 부르는 공간

| 페르소나 | 설명 |
|---------|------|
| **지친 직장인** | 퇴근 후 조용히 하루를 돌아보고 싶은 사람 |

| **기록하는 사람** | 일기·사진·짧은 글로 일상을 남기는 사람 |

공통점: "조용하지만 꾸준히" 활동하는 사람들.

## 3. 주요 기능

### 3.1 오늘의 소소함 (메인 피드)
- 하루 **최대 1개** 글만 작성 가능 (의도적 제한)
- 텍스트 + 사진 1장 구성 (심플)
- 글자 수 제한: 300자 이내
- 좋아요 대신 **"공감해요"** 단일 리액션
- 관심 키워드 중심의 피드 & 사용자를 60%정도 노출하고 나머지 다양한 관심사 컨텐츠로 채움

### 3.2 계절 게시판
피드와 별도로, 계절(봄·여름·가을·겨울) 단위로 운영되는 느슨한 게시판.

| 게시판 | 성격 |
|--------|------|
| **비움 일지** | 버린 것, 정리한 것, 놓아준 것 |
| **산책 기록** | 동네 산책, 자연 관찰, 날씨 이야기 |
| **소소 레시피** | 간단한 한 끼, 차 한 잔, 간식 |
| **고요한 취미** | 독서, 뜨개질, 필사, 그림, 텃밭 |

### 3.3 편지함
- 랜덤 매칭된 사용자에게 **비동기 편지** 교환
- 1주일에 1통 제한
- 느린 대화의 경험

### 3.4 조용한 챌린지
- 예: "일주일 동안 매일 하늘 사진 찍기"
- 예: "한 달간 안 쓰는 물건 하루 1개 비우기"
- 경쟁이 아닌 **자기 기록** 중심

## 4. 디자인 방향

### 톤 & 무드
- **색감**: 따뜻한 뉴트럴 (아이보리, 베이지, 연한 올리브, 먹색)
- **서체**: 부드러운 세리프 or 손글씨 느낌
- **여백**: 의도적으로 많이 — 콘텐츠 사이 숨 쉴 공간
- **애니메이션**: 최소한. 부드러운 페이드 정도만

### UI 원칙
- 한 화면에 하나의 행동만 유도
- 알림은 하루 1회 이하 (조용한 앱)
- 다크 패턴 없음 — 끊임없이 스크롤하게 만들지 않음
- 팔로워 수, 게시물 수 등 숫자 비노출

## 5. 커뮤니티 문화

### 심소 약속
1. **비교하지 않기** — 남의 일상은 구경이 아니라 공감의 대상
2. **천천히 읽기** — 스크롤 말고, 한 글 한 글 머물러 읽기
3. **적게 말하기** — 댓글보다 마음으로, 공감 한 번이면 충분
4. **꾸미지 않기** — 보정 없는 사진, 다듬지 않은 문장도 괜찮음

### AI 활용 정책

> **"AI는 무대 뒤에서 일하고, 무대 위엔 사람만 선다."**

**적극 도입 (운영 뒷단)**
- 게시물 적합성 검증 — 광고·스팸·혐오 표현 자동 필터링
- 민감 소재 사전 감지 — 정치·종교·논쟁성 콘텐츠 자동 플래그
- AI 생성 콘텐츠 탐지 — AI로 작성된 글·이미지를 감지하여 차단
- 이미지 안전성 검사 — 부적절한 이미지 자동 블러/차단
- AI가 사용자 컨텐츠의 논조를 파악하고 부정적인 경우, "다른 사람의 감정을 신경써주세요" 등 넛지 문구 활용
- AI가 금지된 항목에 대한 접근을 검사하여, 업로드 차단 

**엄격 금지 (커뮤니티 앞단)**
| 금지 항목 | 이유 |
|-----------|------|
| AI 생성 글·이미지 게시 | 사람의 진짜 일상만 나누는 공간 |
| 정치·선거·정당 관련 콘텐츠 | 조용한 공간을 지키기 위해 |
| 종교 포교·논쟁 | 위와 동일 |
| 혐오·차별 발언 | 공감의 공간에 어울리지 않음 |
| 상업적 홍보·광고 | 커뮤니티 순수성 유지 |

위반 시: 1차 안내 → 2차 게시물 비공개 → 3차 활동 제한 (단, 혐오 발언은 즉시 제한)

### 운영 톤
- 공지도 편지처럼 쓴다
- "~하세요" 대신 "~하면 좋겠어요"
- 제재보다 분위기로 자정, 단 금지 소재는 예외 없이 적용

## 6. 수익 모델 (가볍게)

| 방식 | 설명 |
|------|------|
| **심소 후원** | 월 구독 (커피 한 잔 가격). 광고 없는 경험 유지 |
| **계절 굿즈** | 분기별 소량 제작 — 엽서 세트, 다이어리, 차 블렌드 |
| **오프라인 모임** | 소규모 산책 모임, 필사 모임 (참가비) |

핵심: 광고를 넣지 않는다. 조용함을 깨는 수익화는 하지 않는다.

## 7. MVP 범위 (1차)

1차 출시에 포함할 최소 기능:

- [ ] 회원가입 / 로그인 (이메일 기반, 소셜 로그인 최소화)
- [ ] 오늘의 소소함 작성 (1일 1글 제한)
- [ ] 메인 피드 (시간순, 알고리즘 없음)
- [ ] 공감해요 리액션
- [ ] 프로필 (닉네임 + 한 줄 소개만)
- [ ] 계절 게시판 1개 (비움 일지)

2차 이후:
- 편지함
- 조용한 챌린지
- 계절 굿즈 스토어

컨텐츠 개발에 앞서 가장 중요한 기능:
- 컨텐츠 검사 AI
- AI로 생선한 컨텐츠 탐지
